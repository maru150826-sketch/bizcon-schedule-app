# SAMPO QUEST 週別空き時間調整 v13

GitHub Pages + Supabaseで動く、SAMPO QUESTビジコンチーム用の予定調整Webアプリです。

## v13で変えたこと

- 同じ日時の入力をまとめて表示
- 同じ日時の提案も重複表示しない
- 一人だけ短い時間しか空いていない場合は「一部参加」として表示
- 提案カードに「フル参加」「一部参加」「不可」を表示
- 同じ日でオンラインと大学のどちらが集まりやすいかを表示
- 週ごとに表示を切り替えられる
- 確定した作業予定を上の方に表示
- オンライン / 大学 / どちらでも の場所入力に対応

## 使い方

1. 表示する週を選ぶ
2. 自分の名前を選ぶ
3. 空いている日付・時間・場所を入力する
4. アプリが集まりやすい日時を提示する
5. フル参加・一部参加・不可を見て、よさそうな日時を確定する

## GitHub Pagesへの反映

以下のファイルをGitHubリポジトリに上書きしてください。

- `index.html`
- `style.css`
- `app.js`
- `README.md`
- `supabase-schema.sql`

最低限、画面と動作の変更だけなら以下の3つで反映されます。

- `index.html`
- `style.css`
- `app.js`

アップロード後、GitHub Pagesで `Ctrl + F5` を押してください。スマホではURL末尾に `?v=13` を付けると古いキャッシュを避けやすいです。

## Supabase

v11以降の `supabase-schema.sql` を実行済みで、`availability_slots` テーブルが存在していれば基本的に追加SQLなしで動きます。

不安な場合は、このv13の `supabase-schema.sql` をSQL Editorで再実行しても問題ありません。
